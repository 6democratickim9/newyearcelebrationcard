* 구글 폰트 import 하는 방법

  ```html
  <link href ="구글 폰트 복붙">
  <style>
      selector(name){
          font-family및 구글 폰트에서 복사한 소스 붙여넣기
      }
  </style>
  ```

  

* media 활용하기

  ``` html
  <style>
      @media screen and (원하는 조건값){
          묶은 클래스 {
              원하는 변경값
          }
          
      }
  </style>
  --> 원하는 조건값에 따라 화면을 바꾸는데, 묶은 클래스나 인자들만 바뀐다.()
  ```
